About This Website

This website showcases a collection of laptop specifications and pricing information. The data is dynamically loaded from a CSV file and presented in a user-friendly table using Flask and Pandas.

It is designed to help users explore and compare different laptop models based on attributes such as:

  • Brand and Processor
  • RAM and Storage
  • Operating System
  • Graphics Capabilities
  • Price, Ratings, and Reviews

This is a demo project created for learning purposes. Future versions may include filtering, searching, and detailed product pages.
